# 🏦 걱정 은행 (Worry Bank) — 걱정 창구

무슨 걱정이든 다 받아드리는 **창구**예요. 상단 탭으로 두 가지 모드를 오갈 수 있어요.

- 💬 **창구(대화)** — 창구직원 페르소나 **‘또박’** 에게 내 말로 걱정을 쏟아내고 → 따뜻하게 받아주면 → **태워 비우거나(탕감)** **금고에 맡겨(예치)** 홀가분해지는 경험. (무슨 말 할지 막막하면 예시 칩을 눌러 시작)
- 🧾 **명세서(진단)** — 원래 기획의 10문항으로 걱정을 정산 → 예금·적금·대출·부실채권 분류, 최고 이자 걱정, 은행 직원 7인, 사고 패턴 진단, 4단계 상환 플랜까지.

## 무엇을 하나요
1. 창구직원 ‘또박’과 **채팅하듯 걱정을 자유롭게 접수**(여러 개 가능, 카테고리 태그는 선택)
2. 또박이 **판단 없이 따뜻하게 받아줌**(페르소나 응답)
3. 접수한 걱정을 🔥 **태워 비우기**(정리 담당 ‘탕감’) 또는 🏦 **금고에 맡기기**(보안요원 ‘철벽’이 대신 보관)
4. 원하면 💭 **되짚어보기** — 실현확률에 따라 CBT 한 줄 코칭
5. **오늘 비운 걱정 / 금고 보관 수** 집계 + 지점장 ‘든든’의 하루 마감 다독임

금고에 맡긴 걱정은 `localStorage`에 저장되어 다음에 와도 또박이 지키고 있어요.

## 페르소나 & 캐릭터
- **창구직원 ‘또박’** 💁 — 20년 차 베테랑. 어떤 걱정도 판단 없이 받아주는 다정하고 담백한 페르소나. (대화의 주인공)
- **정리 담당 ‘탕감’** 🔥 — 걱정을 태워 비워줌
- **보안요원 ‘철벽’** 💂 — 금고에 맡긴 걱정을 지킴
- **지점장 ‘든든’** 🧑‍💼 — 하루 마감 총평
- **이자요정 ‘불어’** 🧚 — 묵힌 걱정에 이자가 붙는다는 경고(확장용)

## AI ‘또박’ 켜기 (선택)
이미 **API 라우트가 연결돼 있습니다.** 키가 없으면 규칙 기반으로, 키가 있으면 자동으로 AI 응답으로 승격돼요.

1. `.env.local.example` 을 `.env.local` 로 복사
2. `ANTHROPIC_API_KEY` 에 [Anthropic 콘솔](https://console.anthropic.com) 키를 입력 (모델은 기본 `claude-3-5-haiku-latest`)
3. `npm run dev` 재시작 → 창구직원 ‘또박’이 LLM으로 답합니다

동작 구조: 클라이언트 `getTellerReply()` → 서버 라우트 **`app/api/teller/route.ts`** → Anthropic API.
페르소나·안전 규칙은 `lib/bank.ts`의 **`TELLER_SYSTEM_PROMPT`** 에 정의돼 있어요. API 키는 서버에만 있고
클라이언트로 노출되지 않습니다. 호출 실패 시에도 규칙 기반으로 폴백해 대화가 끊기지 않습니다.

## 실행
```bash
npm install
npm run dev
# http://localhost:3000
```

## 스택
Next.js 14 (App Router) · TypeScript · Tailwind CSS

## 구조
```
app/page.tsx               # 서비스 홈 = 창구
app/api/teller/route.ts    # AI 응답 라우트 (Anthropic 호출, 키 없으면 규칙 폴백)
components/WorryBank.tsx    # 채팅형 창구 UI (페르소나 대화)
lib/bank.ts                # 저장/응답 로직 + getTellerReply + TELLER_SYSTEM_PROMPT
.env.local.example         # ANTHROPIC_API_KEY 설정 예시
```

> ※ 본 서비스는 셀프케어 도구이며, 전문 심리·의료 상담을 대체하지 않습니다.
